PowerTests.NET

Application site: http://www.powertests.net

PowerTest .NET is a web based multi-user educational software specially designed for usage in schools and universities as an assessment tool or for web based learning.

Feature rich, the system can be used as a replacement for traditional assessment methods or as a complementary tool.
