CodeBeanz Platform

Learning to code can be easy and fun with the right platform. Just open codebeanz.com and start coding - no need to install any software. Our entire curriculum of lessons and projects is tailored made for kids with activities such as interactive graphics and game creation. On codebeanz students love creating programs and share them with their friends!
1.

Web based platform

With CodeBeanz you'll code and run real games and applications directly in your browser. You don't need to install any software on your local machine. Any Windows, Mac or Chromebook computer is perfect for CodeBeanz.

2.

JavaScript language

At CodeBeanz.com we teach JavaScript - the most used and popular programming language nowadays. Our multi-scene code editor is empowering beginners to type their first line of code as well as advanced users to create multi-scene platform games.

3.

Images and sounds

To make coding fun and engaging, CodeBeanz is providing you with a full library of background images, sprites / characters and sounds that you can use in your games and applications.
